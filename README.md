## 项目简介
一个基于 Vue 3 的音乐学习项目，旨在学习和实践现代化前端技术栈。

## 技术栈
- **框架**: Vue 3
- **构建工具**: Vite
- **开发语言**: TypeScript
- **样式方案**: Tailwind CSS 4
- **动画库**: GSAP
- **组件库**: shadcn-vue
- **图标库**: Lucide、阿里巴巴图标库
- **主题**: shadcn-vue 主题系统

## 后端接口 
- **接口文档**: 采用开源node项目https://github.com/XiangZi7/NeteaseCloudMusicApi
